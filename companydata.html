Data Cleaning/Transformation
World Company Data

In this project we clean the world company data and transform to make it more usable for analysis in SQL Server.

To ensure accurate analysis, a four-step process is followed:

  1. Eliminate Duplicates
  2. Standardize Data Format
  3. Adress Missing Values
  4. Remove Irrelevant or Redundant Columns Before Analysis

Before we remove any duplicates, we create a duplicate table called layoffs_staging to secure initial data table

After we have layoffs_staging table now we search for duplicate data with row number


Now we clearly see, we find a row with the number of repetitions is 2 or even 3 times

we will delete this repetitive data but doing so requires calculations and preparation

Then we will standardize the data, where at this stage we will see that the industry type column is still empty or inappropriate and we will correct this value. Next we will see empty values ​​or missing values ​​from the total_laid_off and percentage_laid_off columns, where the industry column that has content is joined to a column that has no content. Now we have a table ready for analysis.
